CBE Currency Update

A localized technical module to update currencies according to Central Bank of Egypt!

Automatic daily updates and manual

Update currencies using Central Bank of Egypt.

International Currency Exchange rates slightly different from CBE Exchange Rates. Therefore this module to adopt these rates as per CBE. You can automatically specify when to fetch the updates using Scheduled Actions under Settings -> Technical!.

Please don't use this module if you're outside Egypt country. All currency rates are compared to EGP Pound

Very low monthly subscription fees

In order to keep our services running, we choose to use the modern IAP of Odoo. Therefore, from General Settings -> CBE Currency Updater section, you can Buy subscription. And it's maintained by Odoo so your privacy is safe!

Just activate the required currency and leave the rest to the updater!

7 days Free Trial

Want to make sure if the updates are accurate? Simply try out 7 days for free before you subscribe!
